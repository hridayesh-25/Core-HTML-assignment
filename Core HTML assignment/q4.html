HTML tags

  1. h1 => heading tag which defines a heading in the browser.
  2. p => paragraph tag which defines a paragraph in a browser.
  3. b => bold tag which makes the text bold in the browser.
  4. br => line break tag which end the line and starts the content from next line in the browser.
  5. i => italic tag which changes the font of text to italic in browser.
  6. a => anchor tag which uses to create hyperlink in the web page for some website.
  7. ul => creates a list in the web page in which every list item will have a bullet point in front of it.
